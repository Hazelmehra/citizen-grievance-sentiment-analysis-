# Models

Trained model files are not stored in this repository in line with MLOps best practices 
and the project's data security guidelines.

The following model files are generated by running the notebook:

| File | Generated In | Description |
|---|---|---|
| `complaint_classifier_model.pkl` | Week 2 | Logistic Regression model for complaint type classification |
| `tfidf_vectorizer.pkl` | Week 2 | TF-IDF vectorizer for department classification |
| `sentiment_model.pkl` | Week 3 | Sentiment/urgency classification model |
| `sentiment_vectorizer.pkl` | Week 3 | TF-IDF vectorizer for sentiment classification |

To regenerate these files, run the full notebook from top to bottom in Google Colab.
